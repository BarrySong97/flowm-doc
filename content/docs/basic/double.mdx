---
title: 什么是复式记账？
---

复式记账其实很简单，就像**记账时要记两笔账**。

## 基本原理

想象一下，你的钱就像水一样，从一个地方流到另一个地方：

- 钱从哪里来？（**来源**）
- 钱到哪里去？（**去向**）

每一笔交易，都要同时记录这两个方面，这就是"复式"的含义。

## 生活中的例子

### 例子 1：买奶茶

你花了 20 元买奶茶：

- **去向**：餐饮费用 +20 元（钱花在了吃喝上）
- **来源**：现金 -20 元（现金减少了）

### 例子 2：发工资 💰

你收到 5000 元工资：

- **去向**：银行卡 +5000 元（银行卡里的钱增加了）
- **来源**：工资收入 +5000 元（这是你的收入来源）

### 例子 3：还信用卡 💳

你用银行卡还了 1500 元信用卡：

- **去向**：信用卡负债 -1500 元（欠的钱减少了）
- **来源**：银行卡 -1500 元（银行卡里的钱减少了）

## 为什么要这样记账？

1. **不会漏记**：每笔交易都有来源和去向，不容易忘记
2. **账目平衡**：所有来源的钱 = 所有去向的钱
3. **清晰明了**：能清楚知道钱是怎么流动的

## 简单记住方法

记住这个公式：

```
资产 = 负债 + 所有者权益
```

用大白话说就是：

```
你拥有的 = 你欠别人的 + 真正属于你的
```

比如你有 10 万存款，但还欠 6 万房贷，那么真正属于你的财富就是 4 万元。

## 复式记账 vs 普通记账软件

很多人用过记账软件，但大部分都是"单式记账"。让我们看看它们有什么不同：

### 普通记账软件（单式记账）

**记录方式**：只记一笔账

- 买奶茶 20 元 → 只记录"支出 20 元，餐饮类"
- 发工资 5000 元 → 只记录"收入 5000 元，工资类"

**看到的信息**：

- 今天花了多少钱
- 这个月收入多少
- 各个分类的支出

**问题**：

- 不知道钱从哪个账户出去的
- 不知道钱到了哪个账户
- 容易出现账目不平衡

### 复式记账软件

**记录方式**：每笔交易记两笔账

- 买奶茶 20 元 → 记录"现金账户 -20，餐饮费账户 +20"
- 发工资 5000 元 → 记录"工资收入账户 +5000，银行卡账户 +5000"

**看到的信息**：

- 每个账户的具体余额
- 钱的完整流动路径
- 资产负债的真实状况

**优势**：

- 账目自动平衡（不会出错）
- 完整的财务状况
- 能看出资产变化

### 举个具体例子对比

**场景**：你用银行卡买了 100 元的书

**普通记账软件记录**：

```
支出：100 元，图书分类
```

**复式记账软件记录**：

```
银行卡账户：-100 元
图书支出账户：+100 元
```

### 哪种方式更好？

**普通记账适合**：

- 只想知道花了多少钱
- 简单的收支记录
- 不关心资产变化

**复式记账适合**：

- 想了解完整财务状况
- 管理多个银行卡/现金
- 有投资、贷款等复杂情况
- 想要专业的财务管理

### 实际差别在哪里？

用普通记账，你知道：

- "我这个月花了 3000 元"

用复式记账，你知道：

- "我这个月花了 3000 元"
- "现金还剩 500 元，银行卡还有 8000 元"
- "信用卡欠了 2000 元"
- "我的净资产是 6500 元"

复式记账就像给你的钱安装了 GPS，随时知道每一分钱在哪里！

## 总结

复式记账就是把每笔钱的"来龙去脉"都记清楚，让你的账本像个小管家一样，时刻知道家里的钱是怎么流动的！
